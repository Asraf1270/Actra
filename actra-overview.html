⚙️ Getting Started

Up and running
in minutes

Three simple steps to get your local development environment ready.

01
Clone the repository
Download the project and navigate into the directory.
git clone https://github.com/your-username/actra.git
cd actra
02
Setup the Backend
Install dependencies and configure your environment variables.
cd server && npm install

# Create .env file:
PORT=5000
DATABASE_URL=your_postgresql_connection
JWT_SECRET=your_secret_key

npm run dev
03
Setup the Frontend
Install client dependencies and launch the dev server.
cd client
npm install
npm run dev
